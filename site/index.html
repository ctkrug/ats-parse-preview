Columns read across, not down

A two-column layout only reads correctly if the file emits one column at a time. Many do not. We count how often the parser hops between your columns and highlight both of them when it reads across instead of down.

Tables that scramble their rows

Cells emitted down the columns instead of across the rows split every label from its value. We find aligned cell grids and tell you when their order breaks.

Pages with no text at all

An exported-as-image resume looks perfect and parses to nothing. If a page yields no characters, you will see it stated plainly rather than as an empty panel.
